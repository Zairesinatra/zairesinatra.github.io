关于正则表达式

It takes a strong man to save himself, and a great man to save another. -- "The Shawshank Redemption"

Ziyi Xie

Dec 29, 2019 • 7 min read

前言

正则表达式是一种文本模式的匹配工具。通常由普通字符(例如字符 a 到 z )以及特殊字符("元字符")组成的文字模式。

正文

新建正则表达式:

var regexp = /xzy/g; // 开发中常用的字面量新建正则表达式对象在代码编译时就会生成
var regexp = new Regexp("zairesinatra","g"); // 构造函数生成的正则对象在代码运行时生成

使用正则表达式:

正则对象的方法: RegExp对象.方法(string)
字符串对象的方法: string.方法(RegExp对象)

修饰符与元字符:

/pattern/flags
  • 正则修饰符标记于表达式之外，用于指定额外的匹配策略，同时为正则对象增加对应的的属性
修饰符 含义 描述
i ignore - 不区分大小写 将匹配设置为搜索时不区分大小写: A 和 a 无异
g global - 全局匹配 查找所有的匹配项
m multi line - 多行匹配 使边界字符 ^ 和 $ 匹配每一行的开头和结尾
s 特殊字符圆点 . 中包含换行符 \n 默认情况下的圆点 . 是 匹配除换行符 \n 之外的任何字符，加上 s 修饰符之后, . 中包含换行符 \n
  • 元字符即描述正则表达式上下文中的行为的字符
var str = "abcd test@zairesinatra.com 1234";
// \b 匹配一个单词边界，也就是指单词和空格间的位置
// 字符集[\w.%+-]匹配字母数字下划线、特殊符号
// \. 匹配 .
var pattern = /\b[\w.%+-]+@[\w.-]+\.[a-zA-Z]{2,6}\b/g;
document.write(str.match(pattern)); // test@zairesinatra.com

正则表达式方法:

  • RegExp 的 exec 和 test 方法
// 空字符串正则匹配
let kong = new RegExp('')

// 字面量新建正则表达式
let reg1 = /xzy/g;
let test1 = 'xzyxxxzzzyyy';
console.log(reg1.test(test1)); // true
console.log(kong.test(test1)); // true

// 正则构造函数创建
let reg2 = new RegExp('zs','m');
let test2 = 'xzyxxxzzzyyy';
let test22 = 'zsxzyxxxzzzyyyzs';
console.log(reg2.exec(test2)); // null
// 对于调用exec方法后返回的数组具有以下两个属性: input - 整个原待匹配的字符串、index - 整个模式匹配成功的开始位置
console.log(reg2.exec(test22)); // [ 'zs', index: 0, input: 'zsxzyxxxzzzyyyzs', groups: undefined ]

// 如果正则表达式包含圆括号,则返回的数组会包括多个元素.首先是整个匹配成功的结果,后面是圆括号里匹配成功的结果,如果有多个圆括号,他们的匹配成功的结果都会成为数组元素
let reg3 = /^_(x)_(y)$/;
let test3 = '_x_y';
console.log(reg3.exec(test3)); // [ '_x_y', 'x', 'y', index: 0, input: '_x_y', groups: undefined ]
console.log(reg3.exec('')); // 空字符串返回 null
console.log(kong.exec(test3)); // 空字符串正则匹配exec返回 [ '', index: 0, input: '_x_y', groups: undefined ]
  • 字符串对象的 match、search、replace、split 方法
// 字符串对象方法就是字符串写前边

// 空字符串正则匹配
let kong = new RegExp('');

//  match() 方法检索返回一个字符串匹配正则表达式的结果[]
let zsstr = 'zairesinatra';
let reg = /z/;
let greg = /z/g;
let regno = /x/;
console.log(zsstr.match(reg)); // [ 'z', index: 0, input: 'zairesinatra', groups: undefined ]
console.log(zsstr.match(greg)); // 使用g标志,则将返回与完整正则表达式匹配的所有结果,但不会返回捕获组 - [ 'z' ]
console.log(zsstr.match(regno)); // null

// search方法返回匹配成功的位置在整个字符串中的索引,如果没有匹配成功任何字符则返回-1,search方法会自己忽略g修饰符
console.log(zsstr.search(reg)); // 0

// str.replace(‘搜索模式’,'替换的内容’);
// 正则表达式不带g修饰符,那么replace方法会替换第一个匹配成功的结果,如果加上了g修饰符,那么会替换所有匹配成功的值
let strr = 'abcdea'
let zy = new RegExp('a')
let zyy = new RegExp('a','g')
console.log(strr.replace('a','o')); // 返回新字符串 - obcdea
console.log(strr); // abcdea
console.log(strr.replace(zy,'o')); // obcdea
console.log(strr.replace(zyy,'o')); // obcdeo

// replace方法的第二个参数可以使用$符号,用来指代所要替换的内容
let params = "[$`-$&-$']"
console.log('abc'.replace('b',params)); // a[a-b-c]c
// \s匹配任何空白字符串 \S匹配任何非空白字符
// \w匹配字母数字下划线(不包括符号)
console.log('hi zairesinatra'.replace(/(\w+)\s(\w+)/,"$2 $1")); // zairesinatra hi

// replace方法的第二个参数也可以是函数,用于将第一个参数匹配到的值替换为函数返回值
let repparams = '2 and 4 square'.replace(/[0-9]+/g, function(num) {return num * num})
console.log(repparams); // 4 and 16 square

// 函数也可以由多个参数.它的第一个参数是整个匹配的内容,第二个参数是组匹配(这时有多少个组匹配就有多少个参数)
let field = {
    "pos_1": "dc",
    "pos_2": "fullstack",
    "pos_3": "hardware"
}
let template = '<span id="pos_1"></span><span id="pos_2"></span><span id="pos_3"></span>'
let zsfunc = template.replace(/(span id=")(\w+)(">)(<\/span>)/g, function(match, $1, $2, $3,$4) { return $1+$2+$3+field[$2]+$4 })
console.log(zsfunc); // <span id="pos_1">dc</span><span id="pos_2">fullstack</span><span id="pos_3">hardware</span>

// split(‘字符串的分割正则’,‘返回数组的最大成员数’)；返回分割后各部分组成的数组
let str = 'a,b,c,d';
let strrr = 'a, b,c,d ';
console.log(str.split('')); // ['a', ',', 'b', ',', 'c', ',', 'd']
console.log(str.split(',')); // [ 'a', 'b', 'c', 'd' ]
console.log(strrr.split(/,*/)); // [ 'a', ' ', 'b', 'c', 'd', ' ' ]
console.log(strrr.split(/, */)); // [ 'a', 'b', 'c', 'd ' ]
console.log(strrr.split(/, */, 3)); // [ 'a', 'b', 'c' ]

正则表达式应用:

// 1.连字符转驼峰
let str = "life-goes-on";
let reg = /-\w/g; // 匹配横杆以及之后的一个字符，全局匹配
// 注意这里不能是/^-\w$/g,会输出life-goes-on,因为^和$分别匹配字符串的开始位置和输出位置导致匹配不到
console.log(str.replace(reg,function($1){
    return $1.slice(1).toUpperCase()
})) // lifeGoesOn

// 2.匹配二进制
var strr = "100110011";
// [xyz]字符集合-匹配所包含的任意一个字符.例如 [abc] 可以匹配 "plain" 中的"a"
var regg = /^[01]+$/g;
console.log(regg.test(strr)); // true

// 3.非负十进制正整
var num = 66;
var numm = '+666';
var zsreg = /^\+?[1-9][0-9]*$/;
var zsregg = /\+?[1-9][0-9]*/g;
console.log(zsreg.test(num)); // true
console.log(zsreg.test(numm)); // true
console.log(zsregg.test(8888)); // true

// 4.匹配十二月
let yue = '10';
let yuereg = /^(0?[1-9]|1[012])$/g;
console.log(yue.match(yuereg)); // [ '10' ]

// 5.匹配QQ账号-最短5最长11
let qq = '11111111';
let qqreg = /^[1-9][0-9]{4,10}$/g;
console.log(qqreg.test(qq)); // true

// 6.匹配常见的固定电话号码
let hnum = '0731-11111111';
let hreg = /^\(?0\d{2,3}[) -]?\d{8}$/g;
console.log(hnum.match(hreg)); // [ '0731-11111111' ]

// 7.ip地址
let ipp = '127.0.0.1';
var ipreg = /((([01]?\d{1,2})|(2([0-4]\d|5[0-5])))\.){3}(([01]?\d{1,2})|(2([0-4]\d|5[0-5])))/g
console.log(ipp.match(ipreg)); // [ '127.0.0.1' ]

// 8.匹配尖括号<>及里面所有的字符
var str66 = "<a herf='www.zairesinatra.com'>";
// ^在方括号表达式中使用,此时它表示不接受该字符集合
var reg66 = /<a[^>]+>/g; // ^> 非 >
console.log(str66.match(reg66)); // [ "<a herf='www.zairesinatra.com'>" ]
