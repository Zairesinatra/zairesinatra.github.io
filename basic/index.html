基础储备

ECMA是一个将信息标准化的组织。而JavaScript是该标准的最流行实现。而语言标准是ECMAScript。

ES2015里函数参数默认值：

代码示例：

// 函数的参数默认值
function sum(x=1,y=2,z=3){
    return x+y+z
}

// 没有JS内置对象arguments对象写法
function summ (){
    var x = arguments.length > 0 && arguments[0] !== undefined ? arguments[0]:1;
    var y = arguments.length > 1 && arguments[1] !== undefined ? arguments[1]:2;
    var z = arguments.length > 2 && arguments[2] !== undefined ? arguments[2]:3;
    return x+y+z
}

console.log(sum(4,2))  // 9
console.log(summ(4,2));  // 9

上述能很明显的看到定义了三个传入参数，而实际我们并没有传入z参数运算。但是结果仍然加上了默认的z参数。这是因为JavaScript函数中存在一个内置对象-arguments对象，类似于数组，准确来说是一个伪数组。包含函数被调用时传入的参数。

声明展开和剩余参数：

在ES5中，可以使用apply()函数把数组转化为参数(区分与call())。为此，ES2015中有了展开运算符(...)。举例来说可以定义参数数组，再通过展开运算符展开参数数组作为传参传入。

代码示例：

// 声明展开与剩余参数
let params = [3,4,5]
// ...展开运算符,把数组转化为参数
console.log(sum(...params)); // 12

// apply()函数也可以把数组转化为参数
console.log(sum.apply(undefined,params)); // 12

// 展开运算符可以代替arguments
function restParamaterFunction (x,y,...a){
    return (x+y) * a.length
}
console.log(restParamaterFunction(1,2,"zairesinatra",true,6)); // 9

// 等同于以上代码
function restParamaterFunctionn (x,y) {
    // slice() 方法可从已有的数组中返回选定的元素。
    // [].slice.call(arguments)能将具有length属性的对象转成数组。
    var a = Array.prototype.slice.call(arguments,2)
    return (x+y) * a.length
}
console.log(restParamaterFunctionn(1,2,"zairesinatra",true,6)); // 9

上述代码需要强调一下的是var a = Array.prototype.slice.call(arguments,2)这个代码段。

首先a是一个数组，同时a并不包含全部的传入参数，因为需要去除x、y。
这里需使用构造函数的原型对象上的.slice()方法。即从已有的数组中返回选定的元素。
再通过.call()方法可以指定的 this 值和单独给出的一个或多个参数来调用一个函数。
那么，这里返回的值才是除去了x、y参数的参数数组。

乘方运算符

在数学运算时较为有用，以下三行代码功能等价。其中ES6中引入 ** 运算符。

const area = 3.14 * r * r

const area = 3.14 * Math.pow( r , 2 )

// ** 为ES6引入,进行指数运算
const area = 3.14 * ( r ** 2 )
