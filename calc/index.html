前言

calc对于许多开发者是既熟悉又陌生的一个函数。这里通过应用场景来加深印象。

正文

calc() 是 CSS 函数，允许在声明 CSS 属性值时执行一些计算。

常用在如下场合：<length>、<frequency>, <angle>、<time>、<percentage>、<number> 或 <integer>。

场景

  • 对目标通过设置 padding 进行水平垂直居中，则需要进行计算页面距离与目标数据(页面一半减去盒子一半)。由此可以使目标增加相应 padding 推至理想位置。
* {
  margin: 0;
  padding: 0;
}
div {
  height: 100px;
  width: 200px;
  /* 
  响应式布局视口单位 vw、vh;1vw = 视口宽度的1%;1vh = 视口高度的1%
  vw = view width
  vh = view height
  */
  padding-top: calc(50vh - 100px/2);
  padding-left: calc(50vw - 200px/2);
}

  • 应用于计算渐变角度，这种情况可以批处理大量渐变而仅需修改单个参数值
#grad1 {
  height: 100px;
  /* background-image: linear-gradient(direction, color-stop1, color-stop2, ...); */
  /* direction: 定义渐变效果的起点和方向(或角度) */
  /* color-stop1, color-stop2,...在其间呈现平滑过渡的颜色 */
  /* 其后是一个可选的停止位置(0％ 到 100％ 之间的百分比值，或沿渐变轴的长度值) */
  background-image: linear-gradient(calc(90deg * 1), #FF4D4D, #06B4FE);
}
#grad2 {
  height: 100px;
  background-image: linear-gradient(90deg, #FF4D4D, #06B4FE);
}
#grad3 {
  height: 100px;
  background-image: linear-gradient(calc(90deg * 2), #FF4D4D, #06B4FE);
}
#grad4 {
  height: 100px;
  background-image: linear-gradient(180deg, #FF4D4D, #06B4FE);
}

  • 自动调整元素大小以适应其容器的大小
* {
    margin: 0;
    padding: 0;
}
.zs {
    background-color: red;
    height: 100px;
    width: calc(100% - 100px);
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    
}

结束

calc(expression) 数学表达式。结果会用作属性值。可使用如下运算符：+ - * / ，注意空格。
