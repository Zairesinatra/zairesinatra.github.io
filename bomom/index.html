BOM

浏览器对象模型

Ziyi Xie

Apr 26, 2019 • 5 min read

开发 Web 应用程序的核心 —— BOM 不同于 JavaScript 、 DOM 有 ECMA、W3C 严格的标准，不同浏览器之间存在差异，兼容性较差。

window 对象

window 对象是 BOM 的核心（document 是 DOM 的核心），表示浏览器的实例。在浏览器中有双重身份：ECMAScript 中的 Global 对象、浏览器窗口的 JavaScript 接口。

  • Global 作用域即通过 var 声明的所有全局变量和函数都会变成 window 对象的属性和方法。

  • DOMContentLoaded 快于 winodw.onload 是因为不考虑图片、flash、CSS 等加载完成。页面加载事件在内容加载完成时触发。调整窗口事件在响应式布局里应用较为常见：移动端项目缩小至某尺寸隐藏某些功能块。

window.onload=function(){} == window.addEventListener("load",function(){})
window.onresize=function(){} == window.addEventListener("resize",function(){})

  • window 对象相对于屏幕左侧与顶部位置的 CSS 像素可以通过现代浏览器提供的 screenLeft、screenTop 属性获的。

  • 不同浏览器窗口大小不同。返回浏览器窗口自身大小使用 outerWidth、outerHeight，返回浏览器窗口中的视口大小使用 innerWidth、innerHeight。

  • 浏览器窗口尺寸通常无法完整显示整个界面，在有限的视口中用户通过滚动轮滑查看文档。文档相对视口滚动的距离属性为 window.pageXoffset|pageYoffset 与 window.scrollX|scrollY。
    滚动页面方法 scrollTo() 是滚动到页面某值处，而 scrollBy() 为相对当前视口滚动。

  • window.open() 方法用于导航到指定 URL，也可用于打开新浏览器窗口。此方法接受4个参数：要加载的URL、目标窗口、特性字符串、布尔值。在目标窗口可以为自定义或特殊窗口名（_self、_parent、_top、_blank）其中布尔值表示新窗口在浏览器历史记录中是否替代当前加载页面。

  • 定时器 setTimeout() 表示指定一定时间后执行代码，setInterval() 为指定每隔一段时间执行代码。

  • 系统对话框：alert() 警告框、confirm() 确认框、prompt() 提示框。

location 对象

location 对象提供当前窗口中加载文档的信息与导航功能。

// 既是 window 的属性，也是 document 的属性
console.log(window.location === document.location) // true

假设浏览器当前加载的 URL 为 https://zairesinatra.com:90/xzy/?usr=zs#frontend ，location 对象内容如下。

属性 值 说明
location.hash #frontend URL 散列值、没有则为空字符串
location.host https://zairesinatra.com:90 服务器名及端口号
location.hostname https://zairesinatra.com 服务器名
location.href https://zairesinatra.com:90/xzy/?usr=zs#frontend 完整的 URL
location.pathname /xzy/ 路径或文件名
location.port 90 端口号
location.protocal https: 协议
location.search ?usr=zs 查询字符串
location.origin https://www.zairesinatra URL 源地址、只读 
// 解析查询字符串、并返回一个以每个查询参数为属性的对象的函数封装
window.location.search="?usr=xzy&age=23"
let getQueryStringArgs = function () {
    // substring() 方法返回一个字符串在开始索引到结束索引之间的一个子集,或从开始索引直到字符串的末尾的一个子集
    let zy = (location.search.length > 0 ? location.search.substring(1) : ""), args = {};
    // 把每个参数保存在args字面量对象
    for (let item of zy.split("&").map(haha => haha.split("="))) {
        let name = decodeURIComponent(item[0]), value = decodeURIComponent(item[1]);
        if (name.length) {
            args[name] = value;
        }
    }
    return args
}
let args = getQueryStringArgs()
console.log(args["usr"]); // xzy
console.log(args["age"]); // 23

URLSearchParams 提供标准的 API 方法检查、修改查询字符串。

let str = "?usr=xzy&age=23"
let searchParams = new URLSearchParams(str);
console.log(searchParams.toString()); // usr=xzy&age=24
console.log(searchParams.has("usr")); // true
console.log(searchParams.get("age")); // 23

通过修改 location 对象修改浏览器地址的操作会修改当前加载的页面

location.assign("www.zairesinatra.com") // 立即启动导航到新 URL, 同时浏览器历史记录新增一条记录

navigator 对象

navigator 是客户端标识浏览器的标准，且各浏览器属性不一。开发中最常用是其 Geolocation API。

navigator.geolocation.getCurrentPosition(function(position) {
  console.log(position.coords.latitude, position.coords.longitude);
});

history 对象

当前窗口首次使用以来用户的导航历史记录。不同 window 的 history 不同。

// window.history.go() 沿任意方向导航, 整数参数表示前进或后退多少步
go(1) // 前进一页
go(-1) // 后退一页

// 不同 window.history.go() 中用参数表示跳转历史条目, back() 与 forward() 中参数无效
history.back() // 后退一页
history.forward() // 前进一页

// window.history.length 记录历史条目数
Google-a-b-c-d // window.history.length == 5

// 历史状态管理 history.pushState() 接受三个参数: state对象、新状态的标题、(可选的)相对URL
let stateObject = {foo:zs}
history.pushState(stateObject, "zairesinatra", "zs.html")
// 完成时状态信息推入历史记录、地址栏改变、浏览器不会发送服务器请求
// pushState()会创建新的历史记录,启用"后退"按钮
