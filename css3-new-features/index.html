HTML5

HTML 泛指 v4，XHTML 可以理解为 XML 风格的 HTML4，即标签关闭、属性小写且被包裹引号、id 代替 name 属性。HTML5 从表现上简化了文档类型声明 <!DOCTYPE html>、字符编码 <meta charset = "utf-8"/>、标签也在不区分大小写之余允许属性值不加引号。

  • 元素拖放

在 HTML5 中，给元素添加一个 draggable 属性并设置该属性值为 true，就能实现元素的拖放。但此时该元素只具备可拖曳的特点，并不能改变元素的位置。改变位置还需要结合触发事件（源元素、目标元素） ondragstart、ondrag、ondragend 之流进行操作。同时如果需要在元素拖放中实现数据传递，则需要使用 dataTransfer 对象在源元素与目标元素之间传递数据。dataTransfer 对象有两个最重要的方法：setData()和getData()。在整个拖曳过程中，开始拖放源元素时（ondragstart 事件），调用 setData() 方法保存数据；然后在放入目标元素时（ondrop 事件），调用 getData() 方法读取数据。

  • Canvas

HTML5 有两个主要的 2D 图形技术：Canvas 和 SVG。Canvas 是使用 JS 动态生成，而 SVG 是使用 XML 静态描述。Canvas 基于位图，适用于像素处理和动态渲染，图形放大会影响质量，但是 SVG 是基于矢量的，图形放大不会影响质量。也就是说，使用 Canvas 绘制出来的是一个位图，而使用 SVG 绘制出来的是一个矢量图。每次发生修改，Canvas 需要重绘，而 SVG 不需要重绘。Canvas 与 SVG 的关系，简单来说就像美术与几何的一般。

  • Web Worker

浏览器没有响应往往是因为某段 JavaScript 代码长时间运行（死循环），导致后面的任务无法执行。H5 中提供多线程解决方案，即通过 Web Worker 创建后台线程执行某段耗时较长的 JS 程序，保证页面正常响应。

在当前 JavaScript 主线程中，使用 Worker() 构造函数新建一个 worker 实例加载某段 JS 文件，发送给一个后台线程处理。

// 新建worker实例
var worker = new Worker('worker.js'); // 参数url表示需要发送到后端线程处理的JavaScript文件的路径
worker.postMessage("zszszs"); // 向后端发送数据给worker线程 => 跳入后端
// 接收后端处理完成的数据
worker.onmessage = function(e){
    console.log(e.data) // e.data￼获取发过来的数据 szszsz
};

// 后端 worker.js
onmessage = function(e){
    // 通过e.data获取前端发送来的数据
    var str = e.data.split("").reverse().join("");
    postMessage(str);
};

CSS3

CSS3 新技术包括：新选择器、文本|颜色|边框|背景样式、2D变形|过渡|动画、多列|弹性布局等。由于新增的一些属性尚未成为 W3C 标准，因此这部分属性在使用上需要携带具有特定内核浏览器的自身私有前缀属性。如果要求 IE6~IE8 兼容则通过 ie-css3.htc 进行实现。

私有前缀 对应浏览器
-webkit- Chrome、Safari
-moz- Firefox
-ms- IE
-o- Opera

  • 多列布局——column

在 CSS3 之前要设计类似报纸那样的多列布局，有两种方式可以实现：一种是“浮动布局”，另一种是“定位布局”。不过这两种方式弊端都很多：浮动布局比较灵活，但不容易控制；定位布局可以精准定位，但是却不够灵活。

CSS3 新增了一种布局方式——多列布局。使用多列布局，可以轻松实现类似报纸、瀑布流般的布局，而且这种布局的自适应能力也非常好。

属性值 说明
column-count(依据 column-width 的 auto 或正整数 n) 多列布局列数
column-width(依据 column-count 的 auto 或正整数 n) 多列布局每列宽度
column-gap(依据浏览器默认长度的 nomal 或px、em、百分比) 两列之间距离
column-rule(复合属性，由3个子属性 column-rule-width\style\color 组成) 两列之间边框样式
column-span(不跨列的 none 和跨所有列的 all) 定义跨列样式

瀑布流布局的特点是每列的宽度是相同的，但是高度是随机的，所以只需要定义 column-width 跟每列的 width 相等，就可以实现理想的效果。部分场合也可以结合在子容器中设置 break-inside 防止多列布局的分页和多区内容中断。媒体查询的具体设备适配这里不再赘述，项目需要时可以查询国内京东凹凸实验室的代码规范。

  • 弹性盒模型

为解决传统布局的死板以及不足，CSS3 新增了新型的弹性盒模型。通过弹性盒模型，可以轻松地创建自适应浏览器窗口的流动布局以及自适应字体大小的弹性布局，使得响应式布局的实现更加容易。

属性 描述
flex(默认值为 1 0 auto) flex-grow、flex-shrink、flex-basis的复合属性
flex-flow(默认值为 row nowrap) flex-direction、flex-wrap的复合属性
order 定义子元素排列顺序
justify-content(flex-start|center|flex-end|space-between|space-around) 子元素在横轴对其方式(间隔都在中间)
align-items(flex-start|center|flex-end|baseline|stretch) 子元素在纵轴对其方式(间隔会环绕)
..................

弹性盒模型成立的前提条件是必须为父元素定义 display:flex; 或 display:inline-flex; 属性，这样父元素才具有弹性盒子模型的特点。在弹性盒模型中，子元素宽度之和小于父元素，则子元素最终宽度就是原来定义的宽度；如果说子元素宽度之和大于父元素，那么子元素会按比例来划分宽度。

// 垂直居中-flex
#father{
    display: flex;
    justify-content: center;
    align-items: center;
    width: 200px;
    height: 160px;
    border: 1px solid silver;
    }
#son{
    width:100px;
    height:50px;
    background-color:hotpink;
    }

        // -----圆形-----
        var cc=document.getElementById("myCanvas3");
        var cxk=cc.getContext("2d");
        cxk.fillStyle="#FF0000"; // 注入红色
        cxk.beginPath(); // 清空子路径列表来启动新路径,当要创建新路径时调用此方法
        cxk.arc(70,18,15,0,Math.PI*2,true);
        cxk.closePath();
        cxk.fill();
        // -----渐变-----
        var cdcd=document.getElementById("myCanvas4");
        var cxtcdcd=cdcd.getContext("2d");
        var grd=cxtcdcd.createLinearGradient(0,0,175,50);
        grd.addColorStop(0,"#06B4FE");
        grd.addColorStop(1,"#00B26A");
        cxtcdcd.fillStyle=grd;
        cxtcdcd.fillRect(0,0,175,50);
        // --图像放置画布--：
        var cqqq=document.getElementById("myCanvas5");
        var cxtqqq=cqqq.getContext("2d");
        var img=new Image()
        img.src="https://res.cloudinary.com/dzb9ldnvl/image/upload/v1622656923/blog/nonocat_feqccv.webp"
        cxtqqq.drawImage(img,0,0);
    </script>
</body>
</html>

在 CSS2 中，可以使用 overflow 属性来定义内容超出元素大小时应该如何处理。而 CSS3 中新增的 overflow-x 属性用来定义内容超出元素宽度时应该如何处理，overflow-y 属性用来定义内容超出元素高度时应该如何处理。这两个属性都有4种取值：visible、hidden、scroll、auto。
