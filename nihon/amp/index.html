日语组成成分

罗马音是提供给外国人供学习日语使用，标注假名发音。

平假名是日语中表音文字的一种，是从中文汉字的草书演化而来的，标注汉字构成日语单词。

日汉字借鉴汉字写法与意思，不借鉴发音，即同字不同音，主要有简体、繁体、和字三种形式。

五十音图的四十六个平假名标注日汉字；四十六个片假名标注外来文。

あ（安） た（太） ま（馬） => a ta ma => 頭

Yama neko sennsei => 山 猫 先生 => 简体和中文的写法和意思一模一样

de nnwa he nn tai => 電話 変態 => 繁体字

tsu ji na gi => 辻 - 十字路口 凪 - 风平浪静 => 和字是中文东拼西凑产生的

a i su ku ri mu => ア（阿）イ（依）スク（久）り（立|利）一ム（牟）　=> 片假名 => 用于英语并借鉴了中文偏旁

日语发音特征 => 短、小、快 => 发音短、嘴型小、语速快
